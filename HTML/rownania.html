Przypominamy, że liczba spełnia równananie, gdy dla tej liczby wartość wyrażeń po lewej stronie i prawej stronie równania są równe.

Sprawdź, które z liczb: 4, 3, -5 spełniają podane rónanie.
x^3 + x^2 = 20x

dla x = 4:
L = 4^3 + 4^2 = 80 P = 20 * 4 = 80
L = P

dla x = 3:
L = 3^3 + 3^2 = 36 P = 20 * 3 = 60
L =/= P

dla x = -5:
L = (-5)^3 + (-5)^2 = -100 P = 20 * (-5) = -100
L=P

Rozwiązać równanie to znaczy znnaleźć wszystkie liczby, które spełniają to równanie.
Rozwiązując równanie, zapisujemy coraz prostrze równania równoważne danemu.

Rozwiąż równania:

a) 7x - 3 = x - 2(5x - 1)
7x - 3 = x - 10x + 2
7x - 3 = -9x + 2 | +9x + 3
16x = 5 | : 16
x = 5 : 16

b) x : 3 - (x - 3) : 6 = 4 | * 6
2x - (x - 3) = 24
2x - x + 3 = 24
x + 3 = 24
x = 21

Niektóre równania z jedą niewiadomą nie mają rozwiązań. Takie równanie nazywamy sprzecznym.
Można również zapisać równania, które spełnia każda liczba rzeczywista. takie równanie nazywamy tożsamościowym.
