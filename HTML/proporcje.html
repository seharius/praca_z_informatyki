Równość dwóch ilorazów nazywamy proporcją

a : b = c : d | czytamy: a ma się do b jak c do d; zakładamy, że b i d są różne od zera

W każdej proporcji iloczyn wyrazów skrajnych jest równy iloczynowi wyrazów środkowych

Jeżeli a : b = c : d to a * d = b * c.

przy rozwiązywaniu równań, które mają postać proporcji, będziemy korzystać z powyższej własności.

2 : x = 3 : (2x + 1) | 2 i 2x to wyrazy skrajne, 3 i x to wyrazy środkowe; zakładamy, że x nie jest równe 0 i 2x + 1 również nie jest równe 0
2(2x + 1) = 3x
4x + 2 = 3x | Iloczyn wyrazów skrajnychjest równy iloczynowi wyrazów środkowych.
